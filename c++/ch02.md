### a162: 多項式計算 (Polynomial)
#### 題目概述
寫出可以計算兩個多項式相加的程式，來驗證自己多項式的作業是否都有計算正確
#### 解題思路
1. 設大於題目範圍數量的陣列(這裡設了1001個位置，分別代表0~1000次方)
2. 分別用na,nb控制a多項式和b多項式的項數
#### 程式碼
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
	int na,nb,d,e;
	int a[1001]={0},b[1001]={0},c[1001]={0};
	cin>>na;
	for(int i=0;i<na;i++){
	    cin>>d>>e;
	    a[d]=e;
	}
	cin>>nb;
	for(int i=0;i<nb;i++){
	    cin>>d>>e;
	    b[d]=e;
	}
    for(int i=0;i<1001;i++){
	    c[i]=a[i]+b[i];
	}
	int m=0;
	for(int i=1000;i>=0;i--){
	    if(c[i]!=0){
	        cout<<i<<":"<<c[i]<<endl;
	        m++;
	    }
	}
	if(m==0){
	    cout<<"NULL!";
	}
}

```

### a122: 山寨版磁力蜈蚣
#### 題目概述
第一行請輸出最一開始的狀態
第二行開始，輸出「刪去第一項後，全部倒轉的結果」
直到數字只剩下一個為止
* 99 77 66 44 11
* 11 44 66 77
* 77 66 44
* 44 66
* 66
#### 解題思路
1. 使用t、n兩個變數作為開頭結尾的代號
t=2、n=7：從第二個位置輸出到第七個位置
2. 因為要輪流從尾/從頭輸出，所以可以設一個變數加上if語法來控制
#### 程式碼
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
	int n,x=0;
	cin>>n;
	int h=0,t=n-1;
	int a[n];
	for(int i=0;i<n;i++){
	    cin>>a[i];
	}
	for(int i=0;i<n;i++){
	    if(x==0){
	        for(int j=h;j<=t;j++){
	            cout<<a[j]<<" ";
	        }
	        cout<<endl;
	        h++;
	        //t--;
	        x=1;
	    }
	    else if(x==1){
	        for(int j=t;j>=h;j--){
	            cout<<a[j]<<" ";
	        }
	        cout<<endl;
	        //h++;
	        t--;
	        x=0;
	    }
	}

}
```
1. 被註解的兩行是本來的錯誤，如果有那兩行的話頭尾都會被扣掉，每一行都會少兩個數字(理論上只會少一個)
2. 初始要設t=n-1而不是n
例：n=5則陣列會設a[0]~a[4]，第23行執行時會從a[5]開始，但a[5]沒有數字

## 章節統整/心得
1. 和上學期相比多了很多比較最大/最小值的題目，要新增兩個變數儲存最大/最小值，並將這兩個變數預設成一定可以被代換掉的數字
可以設超出輸入範圍的數字或 INT_MIN：整數最小值 / INT_MAX：整數最大值
2. 在一開始設陣列長度的時候可以設一個遠超輸入長度的陣列，才不會有陣列不夠用的問題
